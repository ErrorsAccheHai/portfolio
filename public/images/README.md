# Images Folder

Place your images here. The paths below match the defaults in `data/portfolio-data.ts`.

```
public/
├── images/
│   ├── profile-placeholder.jpg     ← Your profile photo (any square image works best)
│   ├── og-image.jpg                ← Open Graph image (1200×630 px)
│   ├── projects/
│   │   ├── project-1.jpg
│   │   ├── project-2.jpg
│   │   ├── project-3.jpg
│   │   ├── project-4.jpg
│   │   ├── project-5.jpg
│   │   └── project-6.jpg
│   └── testimonials/
│       ├── avatar-1.jpg
│       ├── avatar-2.jpg
│       ├── avatar-3.jpg
│       └── avatar-4.jpg
└── resume.pdf                       ← Your resume PDF
```

**Tip:** If an image is missing, the UI shows a graceful fallback
(initials or a gradient placeholder), so the site works immediately
without any images.
